# Report Controls — Index

## When to Use This Reference

Use when adding controls to report bands. This index groups all control types by category. Choose the appropriate reference file based on which control you need.

## Control Categories

### Text Controls
Use [report-controls-text.md](report-controls-text.md) for:
- `XRLabel` — text and data fields
- `XRRichText` — HTML/RTF/DOCX formatted content
- `XRCharacterComb` — fixed-cell character grid (for forms)

### Table Controls
Use [report-controls-table.md](report-controls-table.md) for:
- `XRTable` / `XRTableRow` / `XRTableCell` — structured grid layout with multiple columns

### Image Controls
Use [report-controls-images.md](report-controls-images.md) for:
- `XRPictureBox` — static or data-bound images

### Barcode Controls
Use [report-controls-barcode.md](report-controls-barcode.md) for:
- `XRBarCode` — 35 supported 1D/2D barcode types

### Checkbox Controls
Use [report-controls-checkbox.md](report-controls-checkbox.md) for:
- `XRCheckBox` — checkbox glyphs with optional label text

### Page Info Controls
Use [report-controls-pageinfo.md](report-controls-pageinfo.md) for:
- `XRPageInfo` — page numbers, dates, times, user info

### Chart Controls
Use [report-controls-charts.md](report-controls-charts.md) for:
- `XRChart` — embedded chart control

### Pivot/Cross-Tab Controls
Use [report-controls-crosstab.md](report-controls-crosstab.md) for:
- `XRCrossTab` — pivot table (matrix data layout)

### Sparkline Controls
Use [report-controls-sparkline.md](report-controls-sparkline.md) for:
- `XRSparkline` — compact inline trend charts

### Gauge Controls
Use [report-controls-gauge.md](report-controls-gauge.md) for:
- `XRGauge` — dial or linear gauges

### Layout & Structure Controls
Use [report-controls-layout.md](report-controls-layout.md) for:
- `XRPanel` — container for grouping controls
- `XRPageBreak` — force a new page
- `XRLine` — horizontal or vertical rules
- `XRShape` — geometric shapes
- `XRCrossBandLine` / `XRCrossBandBox` — lines/boxes spanning multiple bands

### Special Controls
Use [report-controls-subreport.md](report-controls-subreport.md) for:
- `XRSubreport` — embed another report's content inline

Use [report-controls-toc.md](report-controls-toc.md) for:
- `XRTableOfContents` — auto-generated TOC from bookmarks

Use [report-controls-pdf.md](report-controls-pdf.md) for:
- `XRPdfContent` — embed PDF pages
- `XRPdfSignature` — digital signature fields

### Common Properties & Expressions
Use [report-controls-common.md](report-controls-common.md) for:
- Common properties shared by all controls
- Expression binding syntax and examples
